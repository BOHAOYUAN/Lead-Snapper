LEADSNAPPER V2.0
STEP 01
Unzip the Package
Extract the downloaded zip archive (LeadSnapper_V2_Release.zip). You will find the following core files inside:
  • desktop_controller.exe — The local background host executable
  • register.bat — Registry script launcher
  • LeadSnapper_Secure.zip — Obfuscated Chrome/Edge extension
STEP 02
Register the Desktop Host
Double-click register.bat in the extracted directory. This automatically registers the local messaging host in the Windows Registry (HKCU) for Chrome and Edge browsers.
🛡️ Local Security Moat: LeadSnapper operates 100% locally. The local registry mapping ensures the Chrome extension can communicate directly with your local python executable.

Open Extension Manager
Open Chrome or Edge browser, and enter the following path in the URL address bar to open the Extensions page:
STEP 04
Load Extension & Connect
1. Turn on Developer Mode in the top-right corner of the Extensions page.
2. Drag and drop LeadSnapper_Secure.zip into the browser to install it, or unzip it and click Load unpacked selecting the folder.
3. Pin the extension. Open the popup, paste your API Keys, and enable Desktop Native Connection. The status badge will turn green showing CONNECTED.
✅ SYSTEM SECURED & CONNECTED
STEP 05
PWA Mobile Remote Setup
1. Keep desktop_controller.exe running on your PC.
2. On your iPhone or Android browser, enter your PC's local IP address (e.g. http://192.168.1.100:8088).
3. Tap the browser share button and click "Add to Home Screen" for a full-screen, native remote command experience.
